🎨 UI Update Prompt

Frontend aesthetic & interaction refinement for React + TypeScript.

🧠 System Context

The UI Update prompt guides Cursor AI in improving visuals and UX of React components without changing logic. It focuses on Tailwind CSS, Framer Motion, and modern component styling.

  • Modify visuals only — keep business logic intact.
  • Use idiomatic React + TypeScript patterns.
  • Maintain accessibility and responsiveness.

🎯 Prompt Template

Goal: ${goal}
Files: ${files}
Design Intent: ${vibe_reference}
Functional Notes: ${functional_notes}
Constraints: ${constraints}

Output Style:
- Return only updated code.
- Add comments explaining UX/visual choices.
- Provide a summary of the changes.

Goal: Add subtle glassmorphism and motion to the dashboard.

Files: `src/pages/Dashboard.tsx`

Vibe: Clean, light, macOS aesthetic.
