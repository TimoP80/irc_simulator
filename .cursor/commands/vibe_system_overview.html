🎛️ 4-Prompt Vibe Coding System

A modular prompt workflow for Node.js, React, and TypeScript projects.

🚀 Introduction

The 4-Prompt Vibe System is a development method created for Cursor AI and modern full-stack stacks such as React, Vite, and Node.js. It introduces a structured, creative workflow that separates intent into four distinct command scopes — each with its own vibe.

💡 Core Philosophy

“Separation of Vibe = Separation of Clarity.”

By splitting coding intents into four scopes, each update remains clean, focused, and maintainable. It prevents AI or human developers from mixing logic, design, and documentation concerns.

✨ Usage Tips

  • Use UI_update for layout & design changes only.
  • Use module_update for logic or data updates.
  • Use doc_update for documentation and explanations.
  • Use project_update for large-scale restructuring.

📘 Example Workflow

When working in Cursor, open the Command Palette and run one of your vibe prompts:

CMD + K → “Run Command” → UI_update.md
CMD + K → “Run Command” → module_update.md

Each prompt defines clear expectations for Cursor’s reasoning and output, giving your AI development a consistent tone and technical focus.
