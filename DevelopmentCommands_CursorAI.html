Cursor AI – Project Update Guide

Use the following commands in Cursor AI to update your TypeScript / React / Node.js project efficiently.

⚠️ Important: After typing a command (like /UI_update), make sure to select it from the command list so it becomes highlighted. If you don’t select it, Cursor AI will treat your input as a generic prompt and the correct template won’t be applied.

UI Updates

For visual or layout changes in your frontend components, use:

/UI_update <describe the change>

Example: /UI_update Update the navbar to include a dark mode toggle

Module Updates

For logic, API, or backend module modifications, use:

/module_update <describe the change>

Example: /module_update Refactor the user authentication module to support JWT

Documentation Updates

For improving or adding documentation, use:

/doc_update <describe the change>

Example: /doc_update Add setup instructions for local development

Project Configuration Updates

For changes to project settings, dependencies, or environment configurations, use:

/project_update <describe the change>

Example: /project_update Upgrade React to version 19 and update dependencies
